
## لازم عشان تشغل بوت 

- [Node.js](https://nodejs.org/en/) v16.6.0 or higher
- [Discord.js](https://discord.js.org) v13
- [Mongodb cluster url](https://www.mongodb.com/) in config.json

## مهم
متنساش يا برو دي
```npm
npm install discord.js
```
